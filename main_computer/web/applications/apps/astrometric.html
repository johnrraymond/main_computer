Controls
  • Drag: orbit the C++ camera.
  • Shift + drag: pan the camera target.
  • Wheel: dolly in/out.
  • Reset Camera: returns to the original Sagittarius A* framing.
  • Start/Stop: controls only the dedicated main-computer-astrometric Compose project.
  • Diagnose: collects container Compose ps/logs/inspect output when the port is closed.
  • Start Port Smoke: diagnostic no-CUDA stream; if it works but GPU does not, the failure is in CUDA/NVIDIA runtime or kernel startup.
